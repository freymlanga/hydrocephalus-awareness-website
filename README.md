# Hydrocephalus Awareness Website

## Description
A simple, responsive informational website developed to raise awareness about hydrocephalus and provide accessible information to the public as well as get funding.

## Technologies Used
- HTML
- CSS
- JavaScript

## Live Website
https://hydrocephalus-theatre-funding-org.netlify.app/

## Purpose
This project was created to support awareness initiatives and demonstrate basic web design and development skills.
